# Twitter-Favorites-Remover
Remove all of your likes.

Twitterのいいねをすべて削除(解除)するツールです

## API制限について
いいねの追加 / 削除は一日3200回までと制限されています

一応ツール側で制限が解除されるまで待つ処理を入れていますが

しばらく経ってからもう一度起動し直すことをおすすめします
